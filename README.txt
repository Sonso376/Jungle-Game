Instruções de como iniciar o jogo:
1. Fazer download de todos os arquivos do programa (.py) e armazena-los no mesmo diretório
2. Executar o arquivo main.py e seguir as instruções no terminal para começar e jogar uma partida.

Informações sobre a AI:
A AI vêm em 4 dificuldades diferentes correspondendo a algoritmos diferentes sendo:
Fácil- Random player
Médio- MinMax com depth reduzida
Difícil- MinMax com AlphaBeta pruning e com deph aumentada
Montecarlo- Utilizando o algoritmo de Montecarlo
